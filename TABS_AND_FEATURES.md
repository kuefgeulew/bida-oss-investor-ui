## ROLE-GATED FEATURES

Investor role: All 22 tabs accessible

Officer role: Only officer-specific tabs (14 tabs in OfficerLayout)

Admin/SuperAdmin role: Only admin-specific tabs (6 tabs in AdminPortalShell)

## MODAL FEATURES

GrievancePortal - Opens as modal from Aftercare tab

FDIEducationModal - Triggered from various discovery contexts

NotificationPanel - Global overlay, triggered by notification icon

IntelligenceLauncher - Quick access panel, available globally

## CONDITIONAL FEATURES

PreArrivalMode - Only shown if investor.status is pre-arrival

InvestorProfileForm - Only shown if profile incomplete

BankReadinessBadge - Updates based on banking setup completion state

NextBestActionPanel - Content changes based on journey progress

## ALWAYS-VISIBLE FEATURES

LanguageSelector - Available in top-right header on all pages

NotificationPanel - Toggle icon in header on all pages

Quick Action Buttons - Available in Overview tab

System Health Indicators - Shown in Overview tab
